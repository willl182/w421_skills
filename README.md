# w421_skills

Coleccion de skills para OpenCode/Claude Code orientadas a desarrollo, documentacion y automatizacion.

## Contenido

Este repositorio incluye skills como:

- `frontend-design`
- `web-artifacts-builder`
- `docx`, `pptx`, `xlsx`, `pdf`
- `mcp-builder`
- `critical-code-reviewer`
- `omarchy`

Tambien contiene referencias, ejemplos y scripts auxiliares para varios flujos.

## Uso rapido

1. Clona el repositorio.
2. Copia o vincula las carpetas de skills en tu ruta de configuracion de OpenCode/Claude Code.
3. Invoca la skill por nombre cuando corresponda.

## Licencia

Cada skill puede incluir su propia licencia y/o terminos de uso en su carpeta.
